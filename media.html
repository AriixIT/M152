Video formats


AVI (Not compatible) - 6.65MB

+ Widely supported

- Not compatible with HTML5


MP4 - 14.3MB

+ Widely supported

+ Great quality

- Known for crashing editing softwares


WEBM - 10.3MB

+ Great for websites

- Decline in popularity



Image formats


PNG - 5.64MB

+ Supports transparency

+ Lossless compression

- Uses a lot of space


JPG - 2.04MB

+ High compatibility and widely spread

- Lossy compression


BMP - 22.8MB

+ Large color spectrum

+ Simple structure

- Takes a lot of space even after compression



Audio formats


MP3 - 80.7KB

+ Lossless compression

+ Widely supported

- Not compatible with CDs


WAV - 517KB

+ Great quality

- Big files


AAC/M4A - 120KB

+ Efficient encoding

- Not too popular and therefore not widely supported
